This site is 100% created from scratch using HTML, CSS3 (Grid, Felxbox etc.), and some Javascript.

As this is just a static site I didn't fell the need to implement it using any fancy framework. Just some lightweight HTML, CSS and Javascript. 😀

It is being served from an Nginx instance running on a Digital Ocean Ubuntu droplet (With both Nginx and the Ubuntu VM being manually configured)

The site is SSL/TLS enabled on the Nginx instance using Let'sEncrypt certs installed with CertBot. The cert is auto-renewed using a CronJob
